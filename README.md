# Twitter App Clone

Twitter App made with React Native and EXPO.

## Installation

```bash
git clone https://github.com/bhattaraijay05/Twitter-App-Clone.git
```

Add firebase configuration files in FirebaseAuth.js

```bash
yarn install
```

## Usage

```
yarn start
```
